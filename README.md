## router-animation

这个分支主要介绍如何在路由切换的时候添加动画效果，使用到 **`@angular/animation`** 库，并且使用到 **Angular V4.2+** 新出的2个属性：

  - **`query`**: 查询
  - **`stragger`**: 交错效果
